1)	Создайте пустую папку на компьютере и клонируйте в нее проект используя следующую команду:
git clone https://github.com/xalera321/cyberclub
2)	Откройте консоль и перейдите в корневую папку проекта используя следующую команду:
cd cyberclub
3)	Выполните команду cd server, чтобы зайти в серверную часть и пропишите команду npm i для установки всех необходимых зависимостей для работы серверной части веб-приложения.
4)	Выполните команду cd client, чтобы зайти в клиентскую часть и прописываем команду npm i для установки всех необходимых зависимостей для работы клиентской части веб-приложения.
5)	Запустите систему:
В консоли по следующему пути: «cyberclub/server» пропишите команду npm run dev для запуска серверной части, а в консоли по следующему пути: «cyberclub/client» прописываем команду npm start для запуска клиентской части.
Открываем браузер и переходим на http://localhost:3000
